
# 📊 Customer Behavior Analysis Dashboard

## 🚀 Project Overview

Understanding customer behavior is essential for making data-driven business decisions.

This project focuses on analyzing customer purchasing patterns, demographics, spending habits, and engagement trends using **SQL** and **Power BI**. The goal is to uncover meaningful insights that can help businesses improve customer retention and increase revenue.

---

## 🎯 Project Objectives

✅ Analyze customer demographics

✅ Identify purchasing patterns

✅ Track customer spending behavior

✅ Discover top-performing customer segments

✅ Generate actionable business insights
